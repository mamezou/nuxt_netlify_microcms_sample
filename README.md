## セットアップ

```bash
# プラグイン をインストール
$ npm install


# サーバーの起動（開発モード）
# localhost:3000
$ npm run dev

# ビルドコマンドなど
$ npm run build
$ npm run start

# 静的データ生成コマンド
$ npm run generate
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
